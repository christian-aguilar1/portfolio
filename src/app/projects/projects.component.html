Proyectos en los que he trabajado

Página CEMET USACH

Página web del Centro de Estudiantes de Metalurgia, USACH.
Aquí se puede visualizar la malla de la carrera, la directiva del CEMET, contactarlos, hacer un reclamo, acceder al drive, ver el calendario de actividades, etc.
Esta página está desarrollada en Angular, con Bootstrap en el diseño y Firebase para el hosting y las bases de datos. Tiene implementado Firebase Auth, firebase database y Realtime Database de Firebase y Firebase Storage.

Studio Ghibli Tracker

Aplicación web para llevar un conteo de las películas de Studio Ghibli vistas por un usuario, además de poder darles una puntuación.
Esta aplicación está desarrollada en React.js en Platzi Master, yo trabajé en el frontend junto con un equipo de 6 personas con la metodología Scrum. Gracias a este proyecto aprendí React desde cero y trabajé de la mano con un equipo backend, un Team Lead y un Technical Expert.
